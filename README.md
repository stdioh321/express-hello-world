## Install dependencies.

```
$ npm install
```

## Run the app

```
$ npm run start
```
![terminal](docs/snap01.png)
<hr />
Open your browser at:

**http://localhost:3000**

![browser](docs/snap02.png)

## Build Docker image

```
$ cd ~/express-hello-world  # or whatever is the directory location of this repo
$ docker build -t hello-world-image .
```
